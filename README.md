# Today I Learned (TIL)
> 새롭게 배우거나 알게 된 지식, 혹은 의미있는 경험을 "배출"하는 곳
---
## 🔥Rule🔥
- 꾸준하게 커밋한다. 하지만 의미없는 커밋은 하지 않는다.
- 배운 것, 알게된 것, 실패한 경험 들을 정리해 커밋한다
- 문서의 경우에는 최대한 "Markdown"으로 정리한다
- 공유할 필요가 없는, 혹은 잘못된 정보를 공유하지 않기 위해 노력한다
- 의미있는 커밋 메시지를 꼭 남긴다
